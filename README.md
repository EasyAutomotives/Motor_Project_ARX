# Motor_Project_ARX
Project using an SMT32F405RGTx microcontroller for motor control via an MMCC (ARX) algorithm. The project predicts the motor's actual and average RPM. The project is still under development and is written in C.

Developed for postgraduate students, specializing in mechatronics technology.
